# RedSavior 📉🛡️ | Learn, Teach, and Save from the Red

**RedSavior** is a specialized social learning platform for traders. In the trading world, "Red" signifies loss. This app was developed to be the "Savior" from those losses by connecting experienced mentors with aspiring learners. 

It combines the real-time engagement of **Twitter** with the deep-dive educational power of **YouTube**, creating a dedicated ecosystem for financial growth.

---

## 💡 The Mission
> "Saving traders from the Red."
In trading, red candles symbolize losses. **RedSavior** provides the tools, community, and educational content needed to turn those red portfolios into green ones through knowledge sharing.

---

## 🚀 Key Features
* **Social Feed (Micro-blogging):** Like Twitter, users can share quick market insights, charts, and tips.
* **Video Learning (Educational Hub):** A YouTube-style interface for mentors to upload deep-dive trading courses and technical analysis videos.
* **Dual User Roles:**
    * **Learners:** Access high-quality trading content to avoid common market pitfalls.
    * **Mentors:** Build a following and teach strategies to the community.
* **Real-time Interaction:** Integrated **Firebase** for instant updates on the social feed.

---

## 🛠 Tech Stack
* **Language:** [Kotlin](https://kotlinlang.org/)
* **Architecture:** Native Android Development (Android SDK)
* **Backend:** Firebase (Real-time Database, Authentication, and Cloud Storage for media)
* **Local Storage:** Room Database
* **UI/UX:** XML & Material Design (Focused on clear, data-driven financial layouts)

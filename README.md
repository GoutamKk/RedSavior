# RedSavior 🩸 | Blood Donation Connect

**RedSavior** is a life-saving Android application designed to bridge the gap between blood donors and those in urgent need. Built during my industrial training at **O7 Services**, this app focuses on real-time data handling, user-friendly UI, and reliable backend integration.

---

## 🚀 Key Features
* **User Authentication:** Secure Sign-up/Login for Donors and Seekers.
* **Real-time Donor Search:** Find available blood donors based on blood group and location.
* **Firebase Integration:** Real-time database management for instant updates.
* **Modern UI/UX:** Clean, intuitive interface built with Material Design.
* **Splash Screen:** Engaging entry point with smooth transitions.

---

## 🛠 Tech Stack
* **Language:** [Kotlin](https://kotlinlang.org/) (Native Android Development)
* **IDE:** Android Studio
* **Backend:** Firebase (Authentication & Real-time Database)
* **Design:** XML, Material Design Components
* **Local Storage:** Room Database (if used) / Shared Preferences
* **Version Control:** Git
